# Экзамен
Программа подключается к БД ,в которой хранится информация о книгах , путем привязки источника данных к DataGridView.
"Поиск" искать книги по: названию книги, авторам и жанрам.
Кнопка "Добавить" добавляет новую строку в ДБ с заранее заполнеными данными, которые можно отредактировать и сохранить нажав кнопку "Сохранить".
Кнопка "Удалить" удаляет данную строку и автоматически сохраняет изменения.
Кнопка "Продать" уменьшает количество книг и автоматически сохраняет изменения.
